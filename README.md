# Câmbio com Python

Um script para pegar os valores, via requisição web, das cotações das principais moedas mundiais:

  - Real
  - Euro
  - Dólar
  - em mais de 170 moedas

# Funções atualmente em funcionamento

  - cambio_dollar(url)
  - cambio_euro(url)
